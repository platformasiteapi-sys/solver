# Деплой TexasSolver API на Timeweb Cloud

Этот гайд поможет тебе развернуть API-обертку на чистом сервере Ubuntu за пару минут.

## 1. Подготовка файлов (Локально)
На данный момент скрипт деплоя ожидает, что папка `d:\Antigravity_projects\m` (содержащая `Dockerfile`, `setup_timeweb.sh` и папку `solver_api`) будет скопирована на сервер.

Ты можешь просто использовать SFTP (например, через FileZilla) или `scp`, чтобы закинуть всю папку `m` на твой Linux сервер в директорию `/root/solver_project`.

## 2. Аренда сервера
1. Зайди в панель Timeweb Cloud.
2. Нажми "Создать сервер".
3. Выбери операционную систему **Ubuntu 22.04** (или новее).
4. Выбери тариф (желательно с 4+ ядрами CPU, так как солвер работает в многопотоке).
5. (Опционально) Закинь свой SSH-ключ для удобного доступа.

## 3. Запуск установки на сервере
Подключись к серверу по SSH, загрузи папку проекта в `/root/solver_project`, как описано в шаге 1, и выполни две команды:

```bash
cd /root/solver_project
bash setup_timeweb.sh
```

**Что произойдет?**
Скрипт автоматически:
- Скачает и установит Docker.
- Скачает официальный Linux-бинарник библиотеки TexasSolver (`console_solver` версии 0.3.3) с GitHub.
- Соберет Docker-образ с нашим Python API и бинарником солвера.
- Запустит контейнер на **80 порту** сервера в фоновом режиме.

## 4. Как тестировать

Когда скрипт закончит работу, он выведет тебе ссылку вида `http://IP-АДРЕС/api/solve`.

Чтобы проверить, что бот сможет работать с сервером, можешь отправить простой запрос прямо со своего компьютера (через PowerShell или Postman):

```powershell
Invoke-RestMethod -Uri "http://<твой-ip-сервера>/api/solve" -Method Post -Headers @{"Content-Type"="application/json"} -Body '{"pot": 50.0, "effective_stack": 200.0, "board": "Qs,Jh,2h", "range_ip": "QQ:0.5,JJ:0.75,TT,99,88,77,66,55,44,33,22,AK", "range_oop": "AA,KK,QQ,JJ,TT,99:0.75,88:0.75,77:0.5", "thread_num": 6, "accuracy": 0.5, "max_iteration": 20}'
```

Сервер должен "подумать" пару секунд и вернуть огромный JSON со стратегией.

## 5. Завершение работы
Когда закончишь сессию игры покера, просто нажми "Удалить сервер" в панели Timeweb Cloud. Поскольку мы используем скрипт, в следующий раз развернуть сервер будет так же просто.
